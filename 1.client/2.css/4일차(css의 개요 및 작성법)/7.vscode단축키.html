visual studio code에서 자주 사용되는 단축키

  1. html문서 템플릿 작성법->html파일 작성후! enter or html:5 enter
  2. 태그 작성하는 방법=>태그 이름 enter=>자동 완성 기능 ex)div enter
  3. 태그를 여러개 생성하는 방법-태그명*n(반복할 수) (ex)div*5
  4. 태그 중첩해서 생성하는 방법=>태그명>태그명2>태그명3 (ex) div>ul>ol*3
        1. 인접한 태그 생성하는 방법=>태그명+태그명2+태그명3=> (ex) div+h1+h2+h3

        2. 아이디 선택자 설정=> 태그명#아이디명 (ex div#name)
        3. 클래스 선택자 설정=>태그명.클래스명 (ex div.test)
        4. 태그{입력하고 싶은 텍스트} 입력후 enter or table (ex) p{안녕하세요}

          안녕하세요

        5. 숫자 삽입(숫자 1씩 자동증가) (ex)div#name$*n(아이디,클래스,텍스트 사용)
        6. ^입력시 상위계층으로 이동(중첩된 상황에서 상위계층으로 이동할때, 여러번 사용가능) (ex) div>ul>li^p

        7. ^^p 한단계 아래로 이동 (ex) div>ul>li^^p

        8. (태그...)그룹으로 묶는 경우=>(ex)div>(ul>li)+(ol>li) 입력 후 enter친 경우
        9. 자동 완성 기능 div#myid.classA.classB 입력하고 enter 친 경우 동시에 div태그에 id와 class선택자를 만드는 방법(div#myid.classA)
